# 🎯 PROJECT OVERVIEW - AutoToko

## Apa itu AutoToko?

**AutoToko** adalah layanan automation untuk toko online menggunakan n8n, yang membantu UMKM mengotomasi operasional mereka dan hemat waktu 10+ jam per minggu.
